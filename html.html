HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar a estrutura e o conteúdo de páginas web, definindo elementos como parágrafos, links, imagens e títulos. Criado por Tim Berners-Lee, funciona como o "esqueleto" de um site, sendo interpretado pelos navegadores, não sendo uma linguagem de programação.

De forma simpls: HTML é a base de qualquer página da web. Ele serve para estruturar o conteúdode um site. uuuui
